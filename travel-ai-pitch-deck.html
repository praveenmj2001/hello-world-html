Personalized Travel Planning. Smart Meetups. Spontaneous Discovery.

The Problem

Travel planning is broken. Let's fix it.

⏱️ Time-Consuming Research

Travelers spend hours researching routes, restaurants, and activities. Information is scattered across dozens of websites.

👥 Group Coordination Complexity

Finding meetup spots that satisfy WiFi, ambiance, timing, and opening hours for diverse groups is a nightmare.

✨ Spontaneity Isn't Easy

Last-minute evening activities? No personalization? Travelers miss hidden gems and serendipitous moments.

Our AI Solution

Elegantly solving all three problems—at once.

🧠 Hyper-Personalization

Per-trip persona wizard captures travel group, style, interests, diet, pets. AI learns and improves suggestions with every interaction via the "heart places" feature.

🎯 Smart Meetup Mode

Tell us location, time, and date. Claude finds quiet, WiFi-friendly venues with real-time opening hours—perfect for teams, families, or dates.

🌙 Spontaneous Drive Mode

Pick your evening vibe (Scenic / Food / City Lights / Nature / Fun / Surprise). AI finds kid-friendly spots within 1hr, open right now. Adventure awaits.

Tech Stack

Built for speed, scale, and security

🎨 Frontend

  • Next.js 16.2.3 (App Router, Turbopack)
  • React 19.2.4 + TypeScript
  • NextAuth v5 (Google OAuth)
  • Tailwind CSS v4
  • Leaflet/OpenStreetMap Maps

⚙️ Backend & AI

  • FastAPI + Uvicorn
  • SQLAlchemy ORM
  • PostgreSQL (Prod) / SQLite (Dev)
  • Anthropic Claude API (64k output)
  • Google Places & Maps APIs

Deployment: Vercel (Frontend) + Railway (Backend) | CI/CD: GitHub Actions with automated security scanning (Bandit, Semgrep, Gitleaks)
